3D Bike Builder
===================

Experimental bike builder project. Core features

- WebGL/ThreeJS based web app
- Supports virtual reality headsets through webvr
- Supports game controllers via Gamepad API

How to run
-------------
- Clone repo
- Run ```bower install```
- Open ```index.html``` in your browser
